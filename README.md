## OpenUtilityPack

Um pacote de utilidades em Java criado para acelerar o desenvolvimento, padronizar tarefas repetitivas e deixar o dia a dia mais suave. O projeto em si nasceu para concentrar ferramentas comuns utilizadas em projetos Java: leitura de arquivos de configuração, conexão com SQL Server e manipulação de arquivos texto. É um toolkit que evolui junto com os meus estudos. Sugestões e melhorias são bem-vindas. E o projeto é livre para o uso e modificação
